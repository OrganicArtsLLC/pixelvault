Pixel Vault — About, Legal & Attribution

What Is Pixel Vault?

Pixel Vault is an educational game museum — a collection of playable HTML5 game prototypes that reconstruct, analyze, and extend the history of game mechanics. Each prototype is a single HTML file, under 50KB, with zero external dependencies.

The project was created by Joshua Ayson (OA LLC) in collaboration with AI (Claude by Anthropic) as an experiment in human-AI creative partnership.

Educational Purpose

This project exists for educational and research purposes:

  • To study and preserve the history of game mechanics
  • To attribute and honor the original creators of classic games
  • To explore the intersection of game design and AI research
  • To demonstrate AI-assisted software development

Intellectual Property Notice

Original Code: All game code was written from scratch through human-AI collaboration. No source code was copied, decompiled, or extracted from any existing game.

Game Mechanics: Game mechanics (rules of play) are not copyrightable under US law. Our prototypes reconstruct mechanics from published descriptions, not from existing code or assets.

Trademarks: Game titles referenced in our metadata and info panels (e.g., "Pac-Man," "Tetris") are trademarks of their respective owners. We reference them for educational attribution only. Our prototypes do not use these names as their own titles.

Visual Assets: All graphics are canvas-drawn geometric shapes. No sprites, images, or visual assets were copied from any existing game.

Not Affiliated: Pixel Vault is not affiliated with, endorsed by, or connected to the original creators or rights holders of any referenced games. All original games belong to their respective owners.

AI-Generated Content

Game code was generated through human direction of AI (Claude by Anthropic). The human provided design intent, creative direction, QA testing, and approval. The AI provided code generation and implementation. This collaborative process is documented in the project's research files.

Privacy

Pixel Vault uses minimal data collection:

  • localStorage: Used only for saving game state (high scores, QA ratings) on your device. Never transmitted.
  • Analytics: When enabled, we use Google Analytics to understand which games are played. No personal information is collected.
  • No accounts required: You can play all games without creating an account or providing any personal information.
