QA & Testing

🔍
QA Dashboard
Visual testing — play each game, rate Certified/OK/Playable/Broken, add notes. Export reports. Re-check badges for fixed games.
http://localhost:8080/tools/qa-dashboard.html
🛡️
Run QA Scanner
Terminal: node tools/qa-pass1.js — checks syntax, game loops, collision, key handling. --fatal-only for quick check.
🔒
Security Scanner
Terminal: node tools/qa-security.js --fix — checks for XSS, eval, external scripts, localStorage, cookies.

Quick Reference

Common Commands

# Start development server
bash serve.sh                           # Port 8080
bash serve.sh 9090                      # Custom port

# Test everything
bash tools/test-games.sh                # Structural tests (460 games)
node tools/qa-pass1.js                  # JS health scanner
node tools/qa-security.js --fix         # Security audit

# Regenerate after changes
python3 tools/catalogue-generator.py    # Catalogue + manifest
python3 tools/lineage-generator.py      # Ancestry graph
bash tools/gen-timestamps.sh            # QA re-check timestamps

# Build for publishing
bash scripts/publish-to-staging.sh --all --dry-run   # Preview
bash scripts/publish-to-staging.sh --all             # Build locally
bash scripts/publish-to-staging.sh --all --sync      # Upload (asks to confirm)
