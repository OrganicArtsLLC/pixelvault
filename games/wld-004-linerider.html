WLD-004 · LINE RIDER

WLD — World Builder Prototype #004 ★★★★☆

How to Play

Draw lines to create a track, then press Space to watch a tiny sledder ride your creation following realistic physics. There are no objectives — this is a pure creative toy.

  • Mouse drag — Draw lines (pencil) or erase (eraser)
  • Space — Toggle between DRAW and PLAY modes
  • P — Select pencil tool
  • E — Select eraser tool
  • C — Clear all lines
  • R — Restart sledder position
  • ? — Toggle this info panel

Tips

  • Start gentle: Draw a slight downhill slope to get the sledder moving.
  • Curves matter: Smooth curves feel better than sharp angles.
  • Momentum: The sledder gains speed going downhill and loses it going uphill.
  • Iterate: Switch back to DRAW mode to add more track, then PLAY again.

History & Lineage

Ancestry: Line Rider (2006, Boštjan Čadež) → Crayon Physics (2007) → various physics sandboxes

Line Rider was created by Boštjan Čadež, a Slovenian university student, as a Flash game in September 2006. It went viral almost instantly — the concept of drawing lines and watching a sledder ride them was irresistible. No score, no timer, no enemies. Just physics and creativity.

The game spawned a massive community of track artists who created elaborate courses set to music, essentially inventing a new art form. Line Rider proved that a game doesn't need objectives to be deeply engaging — the draw-simulate-iterate loop is inherently satisfying.

AI Archaeology

AI Era: 5 — Deep Learning Dawn (2006–2012)

Contemporary AI: Geoffrey Hinton's breakthrough with deep belief networks (2006). GPU-accelerated training beginning. The foundations of modern deep learning being laid.

Hidden Intelligence: Line Rider is a physics simulation sandbox — the player explores a continuous state space through iterative design. Each drawn line changes the dynamics of the entire system. This is essentially manual parameter search in a physics engine, the same kind of exploration that automated hyperparameter tuning would later formalize.

The Convergence: Modern AI can generate optimized Line Rider tracks using evolutionary algorithms or reinforcement learning, but the generated tracks lack the artistic intent and narrative arc that human creators bring. The game is a perfect example of human creativity augmented by computational physics.

If AI Had Designed This: Procedural track suggestions that complete partial drawings, auto-smoothing curves, and difficulty-adaptive starting slopes.

Classification

Series: WLD — World Builder

Line Rider is the purest world builder: the player creates the entire environment, then observes an agent navigating it. No pre-built levels, no constraints beyond physics.

Core Mechanic

Physics sandbox drawing. Draw lines that become physical surfaces. A simulated rider follows gravity, friction, and momentum along your creation. The core loop is draw → simulate → observe → iterate.

What This Prototype Tests

  • Drawing feel — does line creation feel smooth and responsive?
  • Physics satisfaction — does the sledder move convincingly along curves?
  • Iterate loop — is switching between draw/play seamless enough to encourage experimentation?
  • Emergent complexity — do simple lines create surprising physics outcomes?

Prototype Notes

This is a sketch rated 4/5. Simple physics model with gravity, surface sliding, friction, and momentum. Drawing creates line segments that the sledder collides with. Eraser removes nearby segments.
