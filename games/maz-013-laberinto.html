MAZ — Maze Prototype #013 ★★★★☆

How to Play

Navigate the maze to reach the exit. At knowledge gates, answer trivia questions to open the path forward.

  • Arrow keys — Move through the maze
  • 1 2 3 4 — Select answer (A, B, C, D)
  • Correct answers open the gate ahead
  • Wrong answers block the gate and spawn a hazard behind you
  • Reach the green exit to complete each maze
  • 6 mazes with increasing complexity
  • Score = time remaining + correct answers bonus
  • R — Restart current maze
  • ? — Toggle this info panel

History

Cuban educational software developers in the 1990s created games that merged learning with play under resource constraints. The Laberinto del Saber concept — where knowledge literally opens doors — emerged from a tradition of educational maze games stretching back to early microcomputer software. The idea that correct answers physically transform the game space gives trivia genuine spatial stakes, turning abstract knowledge into a navigational tool.

AI Archaeology

AI Era 3 — Expert Systems

The knowledge-as-key mechanic maps naturally onto expert system architectures. Each trivia gate functions as a decision node in a rule-based system — the player's knowledge determines which paths through the problem space remain accessible. Early educational software used similar branching logic to adapt difficulty, a precursor to modern adaptive learning systems.

Prototype Notes

Six procedurally-defined mazes with increasing complexity. Twenty-four trivia questions spanning science, geography, history, and math. The knowledge gate mechanic physically transforms the maze — correct answers dissolve walls, wrong answers erect them. Hazards spawned by wrong answers create backtracking pressure, making each question a genuine risk-reward decision. Timer per maze rewards both speed and accuracy.
