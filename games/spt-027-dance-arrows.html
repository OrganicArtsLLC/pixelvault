How to Play

Notes scroll upward from the bottom. Press the matching key when a note reaches the judgment line at the top.

  • Q — Upper-Left (↖)
  • Z — Lower-Left (↙)
  • Space — Center (●)
  • M — Lower-Right (↘)
  • P — Upper-Right (↗)
  • Enter — Start / Select song
  • R — Restart
  • ? — This info panel

Timing grades: Perfect (±40ms), Great (±80ms), Good (±120ms), Miss. Build combos for higher scores!

History

Pump It Up (펌프 잇 업, 1999)

Created by Andamiro in South Korea, Pump It Up was Korea's answer to Konami's Dance Dance Revolution. Instead of DDR's four cardinal-direction panels, PIU uses five panels arranged in an X pattern: four diagonal arrows plus a center panel. This layout creates fundamentally different step patterns — crossovers, spins, and center-holds that are physically impossible on DDR's cross layout.

The game became a cultural phenomenon across Asia and Latin America, with a competitive scene that continues to this day. PIU's "freestyle" culture encouraged players to add their own dance moves between required steps, creating a performance art dimension absent from DDR.

AI Archaeology

Temporal Pattern Matching (AI-ERA 5)

Rhythm games are real-time sequence alignment problems. The player must predict note arrival times and execute inputs within tight timing windows — the same problem solved by dynamic time warping in speech recognition. PIU's 5-panel layout expands the input vocabulary, creating a richer combinatorial space for step chart design. Designing charts is itself a constraint satisfaction problem: patterns must be physically executable, musically expressive, and progressively challenging.
